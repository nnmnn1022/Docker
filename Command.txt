Docker Command

1.  docker ps [-a]
docker 에 돌아가는 모든 컨테이너 확인

2. docker image ls
docker 에 있는 모든 이미지 확인

3. docker system df
docker 내 이미지들 용량 확인

4. docker pull [이미지]
이미지 다운로드

5. docker search [이미지]
이미지 검색

6. docker container run [--name 이름] [-d(백그라운드)] [-p(포트) 80:80] [이미지]
도커에 있는 이미지를 이름 지정, 포트 지정하여 백그라운드에 실행
